Displays

  • block: O display block é um tipo de display que faz com que o elemento seja carregado como um bloco (pense em um parágrafo feito com

    ). Ele inícia sempre em uma nova linha, e a largura desse elemento automaticamente é preenchida com todo o tamanho disponível.

  • inline-block: O display inline-block é um tipo de display que faz com que o elemento seja carregado numa mesma linha. O elemento por si só é formatado em uma única linha. No inline-block, as larguras e alturas (width e height) são respeitadas.

  • inline: O inline é um tipo de display que permite renderizar elementos numa única linha. A diferença dele para o inline-block é que características como largura e altura não o afetam. No inline, as larguras e alturas (width e height) são completamente ignoradas.
