Positions

  • static: O static é um tipo de position que serve para renderizar o elemento na ordem que o mesmo aparece na página. O static é o position padrão de todo e qualquer elemento HTML.

    static

  • relative: O relative é um tipo de position que serve para renderizar o elemento na ordem que o mesmo aparece na página. Além disso, ele permite que você modifique a sua posição a partir da sua posição atual. Ou seja, se quisermos adicionar uma propriedade "left" ou "right" em nosso elemento, o mesmo surtirá efeito. Ou seja, o elemento irá se deslocar seja para a esquerda ou para a direita.

    relative

  • absolute: O absolute é um tipo de position que serve para renderizar o elemento na página independente da posição que ele se encontra no documento. O seu limite será definido através do primeiro elemento ancestral que não seja estático.

    absolute

  • fixed: O fixed é um tipo de position que serve para renderizar o elemento na página INDEPENDENTE de onde ele está inserido no documento. Ele simplesmente não segue as regras, ele só respeita o browser. Um elemento posicionado com fixed será dimensionado no tamanho da janela do browser da pessoa.

    fixed
